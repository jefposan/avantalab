# Escopo V1 — Vendas Mobile

## Entra na V1

### Produtos

- listar produtos;
- buscar produtos;
- cadastrar produto manualmente;
- importar produtos via XLS/XLSX;
- instalar pacote de produtos;
- ativar/desativar produto;
- preço normal;
- preço promocional opcional;
- imagem opcional.

### Pacotes de produtos

- pacote Tridium como primeiro pacote de referência;
- possibilidade de outros pacotes no futuro;
- copiar produtos do pacote para a conta do usuário.

### Clientes

- cadastrar cliente;
- editar cliente;
- buscar cliente;
- telefone/WhatsApp;
- observações;
- histórico de vendas do cliente.

### Vendas

- criar venda;
- adicionar produtos;
- alterar quantidade;
- aplicar desconto simples;
- selecionar cliente;
- escolher forma de pagamento informativa;
- salvar venda;
- status simples:
  - aberta;
  - concluída;
  - cancelada.

### Relatórios

- vendas do dia;
- vendas do mês;
- total vendido;
- quantidade de pedidos;
- ticket médio;
- produtos mais vendidos;
- clientes mais recentes.

## Fica fora da V1

- nota fiscal;
- cupom fiscal;
- integração com SEFAZ;
- controle tributário;
- comissões complexas;
- estoque avançado;
- contas a receber detalhado;
- múltiplos vendedores em uma mesma empresa;
- painel web completo;
- integração com gateways de pagamento.
